KEY DECISIONS
-------------
- USE CPU MetaWorld (not GPU MuJoCo Warp) for paper numbers
  - GPU port only works on 5/10 tasks (grasping physics broken)
  - Reviewer agent strongly recommended CPU for publishable results
  - CPU runs at ~85 sps, all 10 tasks work
  - 288 ARM cores per Clariden node → massive parallelism

- CW paper hyperparams (verified against official code):
  - 4x256 MLP, LayerNorm+tanh first layer, LeakyReLU rest
  - lr=1e-3, gamma=0.99, polyak=0.995, batch=128, buffer=1M
  - EWC lambda=1e4, L2 lambda=1e5, MAS lambda=1e4
  - PackNet retrain=100K steps, prune ratio=tasks_left/(tasks_left+1)
  - 1M steps per task, episode length 200

- CSC best hyperparams (from GPU sweep, 16 configs):
  - gamma_comp=0.01, grad_scale_beta=5.0 → zero forgetting on 4-task subset
  - Also good: gamma_comp=0.05, beta=0.5 → 73.4% perf, 2.1% forgetting
